## 1. Semantic Search System

This project implements a search system that finds results based on meaning instead of exact words.

### What it does
- Converts text into numerical vectors (embeddings)
- Compares similarity using cosine similarity
- Retrieves the most relevant results

### Tools used
- Python
- Sentence Transformers
- FAISS
- NumPy

### Key idea
Instead of keyword matching, the system understands the meaning of text and returns similar content.
